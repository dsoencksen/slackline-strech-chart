# Slackline Webbing Stretch Chart

A simple web app to visualize the stretch characteristics of different slackline webbings under various loads.

Visit the live app: [slackline-stretch-chart.netlify.app](https://slackline-stretch-chart.netlify.app/)

## IFrame Embedding
You can embed the stretch chart in your own website using the following iframe code:

```html
<iframe src="https://slackline-stretch-chart.netlify.app/" style="width: 100%; height: 100%; border: none"></iframe>
```

## Inspired by Slackline Verband Österreich
This project is inspired by and adapted from the stretch chart created by the Slackline Verband Österreich. You can find their original work [here](https://www.slacklineverband.com/know-how/slackline-vergleich-dehnung/).
